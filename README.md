## 開始使用
1. 請確認已安裝 node.js 與 npm
2. clone 此專案至本地
3. 在本地開啟之後，透過終端機進入資料夾，輸入：

   ```bash
   npm install
   ```

   ```bash
   npm run start
   ```

4. 若看見此行訊息則代表順利運行，打開瀏覽器進入到以下網址

   ```bash
   Listening on http://localhost:3000
   ```

## 開發工具
- node.js 16.15.0
- Express 4.16.4
- Express-Handlebars 6.0.6
- Bootstrap 5.2.0
